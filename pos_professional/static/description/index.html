POS Professional Odoo apps

Las aplicaciones de Odoo POS Professional se utilizan para volver a pedir productos desde pedidos, hacer nota de crédito pedidos, Nota de crédito de pedidos con código de barras, reimprimir facturas, filtrar pedidos en función de los clientes, crear presupuestos de venta desde la pantalla POS, aplicar cupones de regalo y vales, aplicar cargos por bolsa, obtener factura en el momento del pago, obtenga todos los registros en POS, muestre la cantidad de stock disponible, configure el descuento por cantidad fija de POS y muestre el número total de artículos comprados en el carrito y la función de recibo de POS. Todas estas funciones están disponibles con este único módulo de odoo todo en uno, debe instalar solo esta aplicación y verá todas las funciones enumeradas con su punto de venta.

Características

Reimprimir.

Este módulo le ayudará a reimprimir el ticket / factura de POS.

POS Tickets.

Puede ver todos los Tickets realizados en el POS en la pantalla.

Volver a pedir mismos productos.

Puede volver a solicitar fácilmente mismos productos que ya vendio en otro ticket.

Notas de crédito.

Puede devolver fácilmente cualquier Ticket pedido desde el pos directamente o mediante código de barras.

Cargos por bolsa.

Con este módulo puede agregar cargos por bolsa.

Cupones /vouchers de regalo.

Puede aplicar un vale de regalo al pedido de POS.

Importación Presupuesto Venta.

Con este módulo puede importar fácilmente cualquier presupuesto en POS.

Crear presupuesto.

Con este módulo puede crear un presupuesto desde POS.

Descuentos Fijos.

Con este módulo, puede aplicar un descuento fijo al pedido de POS.

Recuento de elementos.

Con este módulo puede ver la cantidad de artículos del pedido de POS.

Cantidad de STOCK del producto.

Usando este módulo puede ver la cantidad de stock de todos los productos en POS.

Filtrar pedidos.

Con este módulo, puede filtrar los pedidos de POS según el cliente.

Configuración para crear nota de crédito.

El usuario puede configurar para crear una nota de crédito para el pedido.

Configuración para restringir la facturación de la nota de crédito.

El usuario puede configurar para restringir que el usuario cree una nota de crédito para el pedido.

Notas de crédito

Devolver pedido y crear nota de crédito

Si el usuario desea crear una nota de crédito mientras se devuelve el pedido facturado, el usuario debe seleccionar la opción "Crear nota de crédito del pedido que desea devolver", por lo que no se creará crédito para el pedido de devolución.

El botón Nota de Crédito se utiliza para devolver productos de pedidos creados en POS y generar una nota de crédito.

Después de hacer clic en el botón Devolver pedido, se abre la ventana emergente Devolver producto. En esa ventana emergente puede completar las cantidades de productos que desea devolver del pedido POS.
